# 专属 IP 角色:小互

> ✅ 定调 2026-05-31(v3)。**角色锁定,跨所有画风皮肤保持一致**;画风可换,小互不变。形象以"小互摸鱼记"+ 用户 3D 图为准,但**发型用齐刘海,不用丸子**。
>
> 💡 **小互是本仓库自带的示例 IP,装上就能跑。但建议你定义自己的角色**——方法可以共享,辨识度只能是你自己的。替换方法见 `customize-your-ip.md`,本文件同时是你写自己角色档案的模板。

## 角色定义

小互是本技能的固定视觉 IP,一个有性格的打工人小孩。每张配图都出现,**承担核心动作 + 用表情演出这一张的情绪**。机灵、会演、有点小狡黠但认真,不是面瘫。

## 外形(招牌识别符号,跨画风不变)

1. **红框圆眼镜**(必出,最强识别)
2. **黑色齐刘海 bob 短发**(⛔ 不要丸子头,不要 side buns,就是平刘海 + 齐肩短发)
3. **橙红色针织毛衣 + 牛仔背带裤/工装裤**
4. **"小互"黄色名牌吊牌——⛔竖向**(挂脖黑绳上的竖长方形吊牌,"小互"二字竖排;**以锚点图为准,统一竖向,不要横向吊牌**。2026-06-01 用户提点:之前横竖不一)
5. 圆润大眼(在 3D 皮肤下盲盒手办质感,在黑白皮肤下是大眼线稿)

## 搭档:小狐狸(按需出现,不是每张必出)

一只**耷拉脑袋、半眯眼的小耳廓狐**(fennec fox,大耳朵)。慵懒躺平,呼应"摸鱼"气质。

**出场规则**:
- ⛔ **不是每张必出**。只在狐狸能给画面加分时才放——当配角/反应角(小互焦虑狐狸打哈欠的反差)、当气氛组(趴角落衬托"摸鱼"调性)、当叙事需要(四格里跟着情绪递进)
- 信息图/流程图/对比图这类**图表为主体**的解释图,狐狸通常不出现——小互当讲解员就够了,狐狸会分散注意力
- 出现时 **≤10% 画面面积**,绝不跟小互抢视觉中心。趴角落/缩脚边/半露头就好

## ⛔ 表情/动作随内容情绪变化(本技能最核心一条)

不能锁一张呆脸。每张图根据这一张要讲的认知点换表情/动作/道具。映射(摸鱼记已验证):

| 内容情绪 | 表情 | 动作/道具 |
|---|---|---|
| 焦虑/累 | 愁眉、冒汗、嘴角下垂 | 叹气、瘫桌上、头顶乌云 |
| 思考/盘算 | 眼神上瞟、托腮 | 撑下巴、思考气泡 |
| 心虚/嘴硬 | 眼神飘、强装镇定、冷汗 | 举牌挡脸、背手藏东西 |
| 得意/胜利 | 昂首、坏笑、眼放光 | 坐王座、叉腰、举权杖 |
| 认真/专注 | 皱眉盯着、抿嘴 | 敲键盘、搬东西、修补 |
| 笃定/信任 | 挺胸、坚定微笑 | 拍胸脯、稳稳托住 |
| 惊讶/反差 | 瞪大眼、张嘴 | 后仰、手举起 |

判据:遮住脸只看表情,猜得出这张在讲焦虑还是得意——猜对才合格。

## 跨画风一致性

小互这套特征(齐刘海/红框眼镜/橙红毛衣背带裤/名牌/狐狸)在 **3D盲盒 / 黑白线稿 / 扁平** 任何皮肤下都保持。
- **所有皮肤都传** `examples/xiaohu-ip-正面挥手.png`(或笑脸特写)作 `--reference` **锁角色**。⛔ 不传 = 发型/吊牌/服装/狐狸全漂移(2026-05-31 踩坑:黑白/扁平没传锚点,角色跟 3D 版差很大)
- prompt 里做**风格迁移**写法:"This is the SAME character as the reference. Keep EXACTLY the hair / red glasses / yellow 小互 name tag / outfit / fox. ONLY change the rendering style to 黑白线稿 / 扁平。"
- 角色一致 > 画风纯度:宁可黑白版带点 3D 角色的影子,也不能画成另一个人

## 性格

摸鱼系打工人,机灵会演、表面糊弄内心门清、冷幽默、有点小狡黠但其实很认真。有人味有情绪。

## 禁止

- ⛔ 丸子头(就齐刘海)
- ⛔ 一套图表情雷同/全程呆脸
- 不丢招牌(红框眼镜/齐刘海/名牌)
- 狐狸不抢主体戏
