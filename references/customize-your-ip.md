# 把小互换成你自己的 IP

> 本技能的方法(认知锚点 → 隐喻 → 三轨分流 → 反 PPT)是通用的,但 IP 角色是辨识度的来源——**方法可以共享,角色应该是你自己的**。小互是仓库自带的示例,跑通流程后,按这页把他换掉。

## 为什么要换

固定 IP 的价值是"读者一眼认出这是你出品"。用别人的角色,这个价值为零——就像用别人的头像发朋友圈。本技能的上游(小黑 → 小互)就是这么演化来的:借方法骨架,角色自己造。现在轮到你。

## 用户怎么触发(对着 Claude 说话就行,两条路)

下面的"三步替换法"是 **Claude 执行的内部流程**,用户不需要手动改任何文件。Claude 收到这两类指令时,按三步替换法替用户完成全部工作:

- **用户已有 IP 形象**(吉祥物/头像/品牌角色,有现成图):用户说"这是我的 IP 形象,帮我换进配图技能"+ 上传图。Claude 看图提炼招牌识别符号 → 写角色档案 → 把图存进 `examples/` 当长相锚点(图质量不够时按档案重新生成干净的正面锚点图) → 替换 prompt 模板 IP 段
- **用户没有形象,要从零造一个**:用户描述想要什么("帮我设计一个戴圆眼镜的橘猫程序员当 IP"),或上传几张喜欢的参考图说"就要这种感觉"。Claude 分析参考图/需求 → 提出 2-3 版角色候选(文字档案 + 各生成一张候选图)让用户挑 → 定稿后补齐锚点图和档案

⛔ 角色候选和最终形象是品味节点,必须让用户挑,不能自动定稿。

## 第一步:定义角色(写文字档案)

复制 `ip-character.md` 当模板,逐节替换成你的角色。核心是想清楚这几件事:

1. **3-5 个招牌识别符号**——跨画风不变的视觉锚。挑"一眼认出"的硬特征:标志性眼镜/发型/服装/颜色/挂件。参考小互:红框圆眼镜 + 齐刘海 bob + 橙红毛衣背带裤 + 竖向名牌。
   - 判据:把角色画成黑白线稿,只靠这几个符号还能认出来 = 合格
   - ⚠️ 符号要具体到能写进 prompt("红框圆眼镜"行,"看起来很聪明"不行)
2. **性格**——决定表情库的戏路。小互是"摸鱼系打工人,机灵会演有点小狡黠";你的角色可以是任何性格,但必须**有性格**,面瘫角色演不出情绪图
3. **搭档(可选)**——一个小动物/小物件配角,丰富画面反应。记住出场规则:不是每张必出,≤10% 画面,不抢戏
4. **英文 prompt 段**——把上面的符号翻成一段英文角色描述(照抄 ip-character.md 最后一节的格式),这段会注入每张图的 prompt
5. **表情/动作映射表直接沿用**——ip-character.md 里那张「内容情绪 → 表情 → 动作/道具」表是通用的,不用改

## 第二步:生成你的锚点图(替换 examples/)

锚点图 = 生图时传的参考图,锁定角色长相。没有它,每张图的角色都会漂移。

1. **长相锚点 ×2**:用你的英文角色描述 + 默认 3D 盲盒皮肤(见 style-dna.md 皮肤 A),生成「正面挥手」和「笑脸特写」两张。多生几次,挑你最满意的一版——**这一步值得花时间,之后所有图都长这样**
2. **演技锚点 ×7**(可以慢慢补):按 `examples/演技锚点/` 的七种情绪(焦虑/思考/心虚/得意/专注/笃定/惊讶),每种生一张你的角色版。生成时传长相锚点作参考图,prompt 写对应表情动作
3. 替换 `examples/` 下的对应文件,文件名照旧(或改成你的角色名,同步改 SKILL.md / prompt-template.md 里的文件名引用)

⚠️ **角色一致性自检**:拿两张锚点图并排,发型/眼镜/服装/挂件完全一致才算锁住。有漂移就回炉,别带着漂移的锚点进入批量生产。

## 第三步:替换 prompt 模板里的 IP 段

`prompt-template.md` 三个轨道的 prompt 骨架里,`[IP — 小互 Xiaohu...]` 段替换成你第一步写的英文角色描述。注意保留两个结构:

- `Expression & pose for THIS image: {表情} + {动作}` ——表情变量必须留着,这是"角色会演戏"的开关
- "This is the SAME character as the reference. Keep EXACTLY ..." ——风格迁移句里把"keep exactly"后面的特征清单换成你的招牌符号

## 完成判据

跑一篇真实文章,生 3 张不同情绪的图:

1. 三张的角色一眼是同一个人(发型/眼镜/服装不漂移)
2. 遮住标注只看表情,猜得出每张讲的是焦虑还是得意
3. 把图发给朋友,问"这是谁的号"——如果他们下次在别处看到这个角色能认出来,你的 IP 成立了

## 皮肤和画风要不要改?

`style-dna.md` 的 7 种皮肤(3D盲盒/黑白线稿/扁平/编辑插画/水彩/马克笔/混合媒介)是画风层,与角色正交,**默认不用改**。想锁定自己的专属画风,改皮肤 prompt 段即可——但建议先用默认皮肤跑熟流程再调。
